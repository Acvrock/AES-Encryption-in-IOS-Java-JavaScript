AES加密 J2EE、IOS、Android、JavaScript 实现 

----
AES 加密,实现 J2EE、IOS、Android、JavaScript 都可用.这里使用的加密配置是: 
> * 使用AES/ECB/PKCS5Padding 模式  
> * 密钥长度：必须为128位16个字符  
> * 密钥偏移量：1234567890123456  
> * 补码方式为PKSC5Padding或PKSC7Padding  
> * 加密结果编码方式为：base64
